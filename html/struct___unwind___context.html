_Unwind_Context::fde

The Unwind Context, is a pointer that is opaque to layers 1 and 2, and is used by layer 3 to store the information needed to do stack unwinding. In our case, this context stores the layer 1/layer 2 exception object, it stores the current register state (which is modified as stack unwinding is performed) and it store the FDE that describes the CFA for the register state that is stored. The FDE will also be changed as the stack is unwound so that it always points to the FDE for the CFA for the currently stored register state.
