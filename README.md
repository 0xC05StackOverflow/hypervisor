# Bareflank Hypervisor

## Description

The Bareflank Hypervisor aims to provide a platform for performing hypervisor research. Some highlights include:

- Zero legacy support (e.g. requires 64bit)
- Written in C++
- Cross Platform
- Few external dependencies
